## 1. 海外华人 / 留学生为什么需要"回国"

如果你长期身在海外，应该早晚会遇到以下至少一类问题：

### 1.1 影音内容拦截

**B站、爱奇艺、腾讯视频、优酷、芒果TV、网易云音乐、QQ音乐** 大量内容仅授权中国大陆，海外 IP 直接访问会看到：

- "由于版权方要求，您的当前地区无法播放该视频"
- "该歌曲因合作方要求，暂时无法收听"
- HTTP 403 / 451 错误页

这不是网络问题，是**服务端基于 IP 地理位置主动拦截**。

### 1.2 国服游戏延迟

**王者荣耀、原神国服、英雄联盟国服、和平精英、永劫无间** 等国服游戏，物理距离决定 RTT：

| 你在哪 | 到中国服务器的最低延迟 |
|---|---|
| 美西（洛杉矶、旧金山） | 150~180 ms |
| 美东（纽约、波士顿） | 230~280 ms |
| 欧洲（伦敦、巴黎） | 250~320 ms |
| 加拿大（多伦多） | 220~270 ms |
| 澳洲（悉尼、墨尔本） | 180~230 ms |
| 日韩 | 50~80 ms |

200ms 以上的延迟对 MOBA、FPS 类游戏几乎不能玩。即使是回合制游戏（原神这类），界面响应也会明显卡顿。

### 1.3 账号风控

海外 IP 登录中国大陆 App，触发风控概率显著高于本地：

- **微信**：被强制安全验证、好友辅助
- **支付宝**：转账被拦截、需补充身份信息
- **淘宝 / 京东**：登录要短信验证，部分商家不发海外
- **银行 App**：直接拒绝海外登录，必须切换中国 IP

### 1.4 跨境 DNS 污染

部分跨境路径上 DNS 解析会被污染或劫持，表现为：

- 域名解析超时 / 解析到错误 IP
- HTTPS 证书报错
- 网页能打开但子资源加载失败

可以用 [china-network-probe](https://github.com/999021-dev/china-network-probe) 一键检测以上四类问题是否出现在你的网络上。

## 2. 回国 vs 翻墙：方向完全不同

容易被混淆，但本质是两个方向相反的需求：

| | **翻墙（出境）** | **回国（入境）** |
|---|---|---|
| 方向 | 中国大陆 → 境外 | 境外 → 中国大陆 |
| 目标 | 访问 Google / YouTube / Facebook | 访问 B站 / 爱奇艺 / 国服游戏 |
| 服务器位置 | 境外 VPS | 国内带海外加速能力的服务商 |
| 主流工具 | Shadowsocks, V2Ray, Trojan | 商业回国加速器，少数自建 |
| 合规性 | 国内灰色 | 国内不直接违法（不绕开监管） |
| 节点供给 | 多 | 稀缺，因为国内带海外能力的 IP 资源有限 |

**关键事实**：境外 → 中国大陆方向的优质 IP 资源**比反方向贵 5-10 倍**。这就是为什么回国加速器普遍比翻墙工具贵的核心原因。

## 3. 不同场景的最佳方案

不同场景对网络的要求差别极大，没有万能方案。

### 3.1 看视频 / 听音乐

**核心需求**：稳定带宽 + 中国大陆 IP（绕版权拦截）

| 方案 | 优点 | 缺点 |
|---|---|---|
| 商业回国加速器 | 即开即用 | 节点共享，高峰期带宽不稳 |
| 浏览器扩展（仅浏览器流量） | 轻量、便宜 | App 流量不走，手机端没用 |
| 自建国内中转 | 独享带宽 | 月成本 $50+，需要技术能力 |

**推荐**：普通用户用商业加速器；重度视频用户考虑自建。

### 3.2 国服游戏

**核心需求**：低延迟 + UDP 优化 + 稳定不丢包

| 方案 | 优点 | 缺点 |
|---|---|---|
| 专门的**游戏加速器**（非普通 VPN） | UDP 优化、专用游戏路由 | 一般只覆盖热门游戏 |
| 普通回国 VPN | 通用 | 对 UDP 不友好，延迟比游戏专用高 |

**关键提示**：游戏延迟 ≠ 网页延迟。即使你检测到网页 100ms，游戏 RTT 可能还是 200ms 以上，因为 UDP 路径和 TCP 路径不同。

### 3.3 支付 / 银行 / 政务

**核心需求**：**稳定的中国大陆 IP**（不是港澳台 IP）

很多人不知道：**中国香港、澳门、台湾 IP 也会被中国大陆服务识别为"境外"**，触发风控。你必须使用**纯中国大陆 IP** 才能通过验证。

| 方案 | 优点 | 缺点 |
|---|---|---|
| 商业回国加速器（北上广深节点） | 简单 | 需要确认是"中国大陆"而非"亚洲" |
| 国内 VPS 自建 | 完全可控 | 国内云服务商对个人海外用户有限制 |

### 3.4 社交（微信 / 微博 / 小红书）

**核心需求**：稳定 IP（避免频繁切换触发风控）

- **不要**频繁切换不同地区的回国节点，会让你的账号被标记
- **不要**用免费 VPN，IP 段被滥用风险高
- **要**选支持"IP 黏性"或"专属 IP"的服务

## 4. 如何选择回国加速器

市面上中文回国加速器（穿梭、快帆、QuickFox、回国 VPN 等）数量越来越多，选择时建议从这五个维度评估：

### 4.1 节点覆盖

- 节点数量：300+ 起步算够用
- 是否能选区：北京、上海、广州、深圳是核心
- 是否避开港澳台节点（除非你明确需要）

### 4.2 协议安全性

- 优选 **TLS 加密**或基于 TLS 的现代协议（VLESS、Hysteria 等）
- 避开 **PPTP、L2TP** 等老旧协议（已知安全风险）
- 加分项：支持 **WireGuard**（速度快，但部分国内 ISP 会限速）

### 4.3 客户端覆盖

| 平台 | 是否支持 |
|---|---|
| Windows | 几乎必备 |
| macOS | 必备 |
| iOS | 看是否在中国大陆 App Store / 海外 App Store 上架 |
| Android | 必备 |
| 路由器 | 加分项（OpenWrt 等） |
| Linux | 加分项 |

### 4.4 支持的场景

- 视频流媒体（B站、爱奇艺）是否专项优化
- 国服游戏（特别是王者、原神、LOL）是否专项加速
- 支付场景是否稳定

### 4.5 价格 vs 试用

- 是否有**免费试用期**或**7 天退款**（直接用 china-network-probe 实测延迟和可达性）
- 月付 / 年付差价
- 多设备授权数

## 5. 自建方案：可行但贵

如果你愿意花时间，自建反向代理也是一条路。

### 5.1 国内 VPS 反向代理

**架构**：海外客户端 → 国内 VPS（阿里云 / 腾讯云）→ 国内目标服务

- **成本**：国内 VPS 月付 ¥30 - ¥200（带宽决定价格）
- **协议**：Shadowsocks、V2Ray、Trojan 等
- **优点**：完全可控、独享带宽、IP 干净
- **缺点**：
  - 国内云服务商对个人用户做反向代理有限制（违反 ToS）
  - IP 被识别后会被服务方封禁
  - 需要持续维护

### 5.2 家用宽带 + 树莓派

如果你国内有家人愿意配合，可以在家用宽带上放一个树莓派做中继。

- **优点**：家用 IP 最干净，几乎不会被风控
- **缺点**：
  - 上传带宽通常很低（10-30 Mbps）
  - 家用宽带 IP 是 NAT 后的，需要打洞或购买公网 IP
  - 国内运营商有动态封端口的策略

### 5.3 自建 vs 商业，怎么选？

| | 商业加速器 | 自建 |
|---|---|---|
| 上手成本 | 5 分钟 | 数小时 |
| 月成本 | $5 - $20 | ¥30 - $50 |
| 稳定性 | 服务商负责 | 自己负责 |
| IP 干净度 | 共享，可能被风控 | 独享，干净 |
| 适合人群 | 90% 用户 | 技术宅 + 重度使用 |

## 6. 常见坑

### 坑 1：免费 VPN

**不要用免费的回国 VPN**。免费 VPN 的盈利模式大多是：

- 卖你的浏览数据
- 把你的设备当作出口节点（你的 IP 在被别人用）
- 弹广告

### 坑 2：宣传"中国 IP" 实际是港澳台 IP

部分加速器宣传"切换到中国 IP"，但实际给的是**香港 / 澳门 / 台湾 IP**。这些 IP 在很多大陆服务（特别是支付、银行）面前**会被识别为境外**，达不到你的目的。

**验证方法**：用 [china-network-probe](https://github.com/999021-dev/china-network-probe) 跑一次，检查"归属地"是否显示"China"（country_code = CN），而不是 HK / MO / TW。

### 坑 3：节点共享导致风控

商业加速器节点共享，意味着同一个 IP 被很多人同时用。如果你登录支付宝 / 银行时这个 IP 上有其他人触发了风控，你也会被连坐。

**对策**：选择支持"独享 IP"或"专属节点"的高级套餐。

### 坑 4：跨平台账号一致性问题

部分用户在手机和电脑用不同加速器节点，导致同一个微信账号短时间内出现"美西 → 中国 → 加拿大"的登录轨迹，触发风控。

**对策**：所有设备用同一个加速服务、固定到同一个节点。

### 关键指标

| 指标 | 期望（加速后） |
|---|---|
| **出口 IP 归属地** | China（CN），而非 HK / TW |
| **平均延迟** | < 100 ms（理想），< 300 ms（可接受） |

| **HTTP 异常数** | 0（关键服务无 403 / 451） |

如果一个加速器**不能让所有指标达标**，那它对你的场景就不是最优选择。

## 小结

- 海外用户访问中国大陆有三类核心问题：**地理延迟 / 地区版权 / DNS 污染**
- **回国 ≠ 翻墙**，是反方向的需求，资源更稀缺、价格更贵
- 选加速器看：节点覆盖、协议安全性、客户端覆盖、场景适配、试用/退款
- 重度技术用户可以自建，但维护成本不低
- **永远用客观数据验证**，不要轻信厂商宣传
